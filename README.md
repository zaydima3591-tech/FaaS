# FaaS
Демонстраційний проєкт до доповіді **«Serverless: міфи та реальність у 2026 році»**. Створений для того, щоб показати, як швидко та безкоштовно підняти бекенд для невеликого вебдодатка без адміністрування серверів.

## 🛠️ Стек технологій
* **Frontend:** Чистий HTML5, CSS3, JavaScript (Vanilla JS)
* **Backend:** Node.js (Serverless Functions / FaaS)
* **Хостинг та деплой:** Vercel

## ⚙️ Як це працює
Проєкт складається з двох частин:
1. Статична сторінка (`index.html`), яка відправляє асинхронний запит до API.
2. Безсерверна функція (`api/hello.js`), яка обробляє запит у хмарі (AWS Lambda під капотом Vercel) та повертає відповідь у форматі JSON.
